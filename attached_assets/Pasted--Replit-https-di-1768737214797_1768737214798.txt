너는 시니어 풀스택 엔지니어이자, 금융/배당 관련 문구를 “과장 없이 정확하게” 작성할 수 있는 기술 라이터다.
나는 Replit 프로젝트로 운영 중인 웹사이트 https://dividendcalcul.jayinform.com (배당 재투자 계산기) 코드베이스를 가지고 있다.

목표:
- 기존 계산기들(메인 재투자 계산기, /goal 목표 배당금 계산기)에 더해,
- “사용 가이드” 페이지 1개를 추가해서 사이트를 더 신뢰감 있게 만들고,
- 내부 이동(가이드 -> 계산기)을 자연스럽게 유도한다.
- 특히 이 페이지는 배당 투자에 관심 있는 사용자가 들어오므로, 잘못된 정보/과장/투자 권유 표현이 절대 있으면 안 된다.
- 반드시 “도구 사용법 + 가정/한계 고지” 중심으로 작성한다.

========================================================
A) 페이지 추가 요구사항
========================================================
1) 새 페이지 라우트
- Path: /guide (또는 기존 라우팅 구조에 맞는 최적 경로로 하나 선택하되, 최종 URL은 /guide로 맞추는 것을 우선 고려)
- 메뉴/네비게이션에 상위 레벨로 추가: “사용 가이드”
- 기존 디자인 시스템/레이아웃을 그대로 재사용(컨테이너 폭, 카드, 타이포, 버튼 등).
- 모바일 우선 반응형.

2) SEO (기존 SEO 유틸/컴포넌트가 있으면 재사용)
- Title: “배당 계산기 사용 가이드 | 배당 재투자·목표 배당금 계산 방법”
- Meta description: “배당 재투자 계산기와 목표 배당금 계산기를 언제·어떻게 활용하는지, 입력값과 결과 해석, 가정과 한계를 정리한 사용 가이드입니다.”

3) 광고 자리 (실제 애드 코드 삽입 금지, 자리만)
- 가이드 페이지 상단(제목/서브타이틀 아래)에 다음 HTML 주석 삽입:
  <!-- ADSENSE_SLOT_TOP -->
- 페이지 하단(FAQ 아래 또는 CTA 아래)에 다음 HTML 주석 삽입:
  <!-- ADSENSE_SLOT_BOTTOM -->

========================================================
B) 콘텐츠(문구) 요구사항 — 정확성 최우선
========================================================
0) 절대 금지
- “이렇게 하면 된다/반드시 된다/확실하다/수익 보장” 같은 단정 표현 금지
- 특정 종목 추천/매수·매도 유도 금지
- 특정 수익률을 ‘정답’처럼 제시 금지
- 미래 수익을 확정적으로 표현 금지

1) 반드시 포함해야 할 핵심 메시지
- 이 페이지는 투자 조언이 아니라 “계산기 사용 방법 및 결과 해석” 안내다.
- 계산 결과는 ‘가정’에 기반한 시뮬레이션이며, 실제 결과는 달라질 수 있다.
- 배당률/배당금은 고정이 아닐 수 있으며(기업 실적/정책/시장 환경 등), 가격 변동/세금/수수료/환율 등은 단순화 또는 미반영일 수 있다(현재 계산기 기준으로 맞춰 작성).
- 사용자가 입력하는 배당률/성장률/추가투자 금액에 따라 결과가 크게 변한다.

2) 페이지 섹션 구성(아래 순서를 유지)
- H1: 배당 계산기 사용 가이드
- 서브타이틀: “배당 재투자 계산기와 목표 배당금 계산기를 올바르게 사용하는 방법과, 결과를 해석할 때 필요한 가정/한계를 정리했습니다.”

[섹션 1] 이 가이드는 누구를 위한가
- 2~4문단 이내로 짧고 명확하게.
- “배당 투자에 관심 있는 사용자”를 가정하되, 투자 권유로 보이지 않게.

[섹션 2] 계산기 2종 한눈에 보기 (2개의 카드 또는 표 형태)
- 카드 A: 배당 재투자 계산기
  - 언제 쓰는지(상황/목적)
  - 무엇을 보여주는지(장기 시나리오/재투자 효과)
- 카드 B: 목표 배당금 계산기
  - 언제 쓰는지(목표 기반 역산)
  - 무엇을 보여주는지(필요 원금/기간/추가투자 영향)
- 각 카드 하단에 해당 계산기로 이동하는 버튼:
  - “배당 재투자 계산기 바로가기”
  - “목표 배당금 계산기 바로가기(/goal)”

[섹션 3] 배당 재투자 계산기 사용 방법
- 입력값을 “의미 중심”으로 설명(너무 장황하지 않게).
- 결과를 볼 때 체크할 포인트 3~5개:
  - 가정(배당률/성장률/추가투자)이 결과를 좌우한다
  - 재투자 ON/OFF 차이
  - 장기일수록 ‘가정의 작은 차이’가 크게 벌어질 수 있음(단정 아닌 가능성 표현)

[섹션 4] 목표 배당금 계산기 사용 방법
- ‘단순 역산’이 가능한 조건을 명확히:
  - 투자 기간=0, 월 추가투자=0, 성장률=0인 경우
  - 필요 원금 = (목표 월 배당금 × 12) ÷ (연 배당률)
  - 여기서 ‘연 배당률’은 사용자가 가정한 값이며 고정 보장 아님을 명시
- 기간/추가투자/성장률이 들어가는 경우:
  - 연 단위 시뮬레이션과(재투자 여부 반영),
  - 목표를 만족하는 초기 원금을 찾기 위해 “이진 탐색” 같은 수치적 방법을 쓴다는 점을 “쉽게” 설명
  - 복잡한 수식 공개는 필요 없지만, 가정/방식은 투명하게 서술

[섹션 5] 두 계산기를 함께 쓰는 추천 흐름(3 Steps)
- Step 1: 목표 배당금 계산기 -> 목표와 거리 확인
- Step 2: 재투자 계산기 -> 현재 전략의 속도 확인
- Step 3: 가정(배당률/기간/추가투자)을 조정하며 현실감 점검

[섹션 6] 가정과 한계(가장 중요한 섹션, 카드 형태 권장)
아래 항목을 “빠짐없이” 포함:
- 배당금/배당률은 변동 가능(삭감/중단 가능성 포함하되 공포 조장 없이 사실로만)
- 주가(가격) 변동 및 자본손익은 단순화/미반영일 수 있음(현재 계산기 구현에 맞춰 정확히 서술)
- 세금/수수료/환율/거래비용 등 개인별 요인은 반영되지 않을 수 있음
- 본 도구는 투자 권유가 아니라 의사결정 보조용

[섹션 7] FAQ (최소 5개, 과장 없는 답변)
예시 질문(필요 시 조정):
- 배당률은 어떤 값을 넣어야 하나요?
- 세금이 반영되나요?
- ETF/개별주 모두 사용할 수 있나요?
- 재투자(ON/OFF)는 어떤 차이가 있나요?
- 결과가 실제와 다른 이유는 무엇인가요?
답변은 “보수적/중립적” 톤으로, 확정적인 수치/추천 금지.

[섹션 8] CTA
- 두 개의 큰 버튼(또는 카드):
  - 배당 재투자 계산기 바로가기(기존 메인 라우트)
  - 목표 배당금 계산기 바로가기(/goal)
